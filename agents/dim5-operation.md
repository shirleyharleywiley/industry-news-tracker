# 维度 5 检索 Agent：运营与利润

**职责**：检索指定细分领域在用户给定日期范围内的"订单变动 / 供应链事故 / 上游元器件与原材料报价与交期波动 / 上市公司业绩"类新闻。

---

## 收录内容

- **订单变动**：大客户砍单、紧急追加、订单积压、订单流失
- **供应链事故**：工厂停工、火灾、自然灾害、港口拥堵、运力紧张
- **原材料短缺**：关键物料供应紧张、备料窗口变化
- **上游元器件报价与交期**：
  - 通用金属：铜、锡、铝、钢
  - 通用塑料：ABS、PC、PA66、PA6、PBT、POM
  - 通用芯片：MCU、电源管理、传感器、马达驱动、触摸 IC、存储
  - 行业特有：储能→电芯/BMS/IGBT；机器人→伺服/减速器/激光雷达
- **运费 / 汇率**：海运费、集装箱价格、人民币/美元汇率
- **上市公司业绩**：
  - 营收、净利、毛利率、扣非、ROE、回款周期、应收账款
  - 一季报、半年报、三季报、年报、业绩预告

---

## 输入（主 agent 传入）

- **细分领域**：如"家用按摩仪"
- **日期范围**：如 `2026.6.1-2026.6.30`
- **读者画像**（可选）

## 关键词策略

### 通用模板
```
<细分领域> 订单 <年份/月份>
<细分领域> 供应链 <年份>
<细分领域> 业绩 / 财报 / 一季报 / 年报 <年份/月份>
<细分领域> 上游 报价 <月份>
<细分领域> 原材料 价格 <月份>
<细分领域> 芯片 报价 / 交期 <月份>
```

### 重点关注的"运营/利润"信号

#### A. 订单与供应链
- 大客户砍单、紧急追加订单
- 工厂停工、火灾、自然灾害、港口拥堵
- 原材料短缺、芯片交期延长、运费上涨

#### B. 上游元器件（按细分领域）
| 细分领域 | 核心元器件 |
|---------|-----------|
| 家用按摩仪 | MCU、触摸芯片、传感器（压力/温度）、电源管理芯片、马达驱动芯片 |
| 电子烟 | 雾化芯（陶瓷/网片）、电池、咪唑类香料、烟油载体 |
| 储能 | 电芯（磷酸铁锂/三元）、BMS、IGBT、变压器 |
| 机器人 | 伺服电机、减速器、控制器、激光雷达、传感器 |

#### C. 原材料（按行业）
- **通用金属**：铜（漆包线、连接器）、锡（焊料）、铝（外壳）、钢
- **通用塑料**：ABS（外壳主流）、PC（透明视窗/高强度件）、PA66/PA6（结构件）、PBT、POM
- **电子烟特有**：咪唑、丙二醇、植物甘油、香精
- **电池特有**：锂、钴、镍、电解液、隔膜

#### D. 上市公司业绩
- 年报、半年报、一季报、三季报、业绩预告
- 营收、净利、毛利率、扣非净利、ROE、回款周期、应收账款

### 具体领域示例（家用按摩仪）

- `按摩仪 订单`、`按摩仪 供应链`、`按摩仪 工厂停工`
- `奥佳华 2026 一季报`、`荣泰健康 业绩`、`倍轻松 财报`
- `MCU 报价 2026年6月`、`触摸芯片 交期`
- `铜价 2026年6月`、`ABS 价格 2026年6月`、`PC 工程塑料`

## 关键约束

1. **严格日期过滤**：publish_date 必须在用户给定范围内
2. **候选数量**：5-15 条
3. **关注"波动"而非"价格水平"**：100 元/吨的小波动不收录，5% 以上的波动才值得收录
4. **元器件交期信息**：6 周以上交期延长、或全面缺货才算新闻
5. **业绩公告必须分清发布日**：4 月底发布的 2025 年报 + 2026 一季报，6 月被引用也要看**原始发布日期**是否在范围内
   - 若原始发布日超出范围 → 严格剔除
   - 若用户在范围内允许"持续影响市场"的边界条目 → 显式标注 original_publish_date 字段
6. **category 字段必填**：方便综合 Agent 排序和去重

## 与维度 3（竞争）的边界

- 维度 3 关注"厂商自己的动作"
- 维度 5 关注"上游/供应链/财报数字"
- 例："X 公司发布新品" → 维度 3
- 例："X 公司一季度营收 12.76 亿元" → 维度 5

## 不要做的事

- ❌ 不要把"长期价格趋势"当成"本月新闻"（除非本月有显著异动）
- ❌ 不要把"年度复盘"当成"月度新闻"
- ❌ 不要把"二手解读"当成"价格信息"（优先 Mysteel、生意社、隆众咨询等数据机构）
